AI start¶

Observation Space¶

The observation consists of a 3-tuple containing: the player’s current sum, the value of the dealer’s one showing card (1-10 where 1 is ace), and whether the player holds a usable ace (0 or 1).

The observation is returned as (int(), int(), int()).

Action Space¶

The action shape is (1,) in the range {0, 1} indicating whether to stick or hit.

0: Stick

1: Hit

Rewards¶

win game: +1

lose game: -1

draw game: 0

win game with natural blackjack: +1.5 (if natural is True) +1 (if natural is False)

Episode End¶

The episode ends if the following happens:

Termination:¶

The player hits and the sum of hand exceeds 21.

The player sticks.

An ace will always be counted as usable (11) unless it busts the player.

class Kane:
    def __init__(self):
        """Initialize a Rule-Based agent for a Blackjack-like game using observation space."""
        pass

    def get_action(self, observation: tuple[int, int, int]) -> int:
        """
        Decides whether to hit or stand based on the current observation which includes
        the player's total, dealer's visible card, and whether the player has a usable ace.
        
        Args:
            observation: A tuple containing the player's current total, the dealer's visible card,
                         and whether the player has a usable ace (1 for yes, 0 for no).
        
        Returns:
            An integer indicating the action to take: 0 for 'stick' (stand) or 1 for 'hit'.
        """
        return self.apply_predefined_rules(observation)
    
    def apply_predefined_rules(self, observation: tuple[int, int, int]) -> int:
        """
        Apply predefined rules to decide whether to hit or stand based on the observation,
        leveraging statistical expected returns.
        
        Args:
            observation: The current state observation as a tuple (player's total, dealer's visible card, usable ace).
        
        Returns:
            An integer indicating the action: 0 for 'stick' (stand) or 1 for 'hit', based on optimized strategy.
        """
        player_total, dealer_visible_card, usable_ace = observation
    
        # Adjust player's total if there's a usable ace and using it as 11 doesn't bust the hand
        if usable_ace and player_total + 10 <= 21:
            player_total += 10
    
        # Decisions based on statistical expected returns
        if player_total >= 17:
            return 0  # Stick on 17 or more, standard strategy
        elif player_total >= 13:
            # Stick if the dealer has a low card (2-6), otherwise hit
            return 0 if 2 <= dealer_visible_card <= 6 else 1
        elif player_total == 12:
            # If the dealer has 4, 5, 6, it's safer to stand; otherwise, take a hit
            return 0 if 4 <= dealer_visible_card <= 6 else 1
        else:
            # With 11 or less, always hit since you can't bust
            return 1

    obs, info = env.reset()
    done = False
    total_reward = 0  # Keep track of total rewards in this episode

    # play one episode
    while not done:
        action = agent.get_action(obs)
        obs, reward, terminated, truncated, info = env.step(action)

        done = terminated or truncated
